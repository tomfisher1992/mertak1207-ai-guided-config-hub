## What It Does

mertak1207.github.io is a web-based tool made for Valorant setup management. It brings agent profiles, crosshair tuning, sensitivity adjustments, and bind organization into one browser-accessible workflow.

The project also offers AI-assisted configuration support to make setup decisions easier and faster. It is built for users who want a clear, repeatable way to maintain multiple game settings without rebuilding everything from scratch each time.

## Main Features

- Keep Valorant agent profiles organized in one centralized workflow
- Set up crosshair preferences for different agents or playstyles
- Modify sensitivity values through a dedicated configuration path
- Create and manage key binds from a single interface
- Use AI-assisted guidance to speed up setup decisions
- Separate profile-based settings for simpler switching between configurations
- Run the tool through a web interface instead of a desktop app
- Apply repeatable configuration updates as your preferences evolve

**Can I adjust crosshair and sensitivity at the same time?**  
Yes. Both are handled within the same setup process.

**Where are the configuration files kept?**

**Is AI-assisted setup mandatory?**  
No. It is available as guidance, but manual configuration is still supported.
